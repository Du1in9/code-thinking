11.1 理论基础

图的概念

分类：有向图是指 图中边是有方向的，无向图是指 图中边没有方向。

加权有向图，就是图中边是有权值的。加权无向图也是同理。

度：无向图中有几条边连接该节点，该节点就有几度。在有向图中，每个节点有出度和入度。

连通：在无向图中，任何两个节点是可以相互到达的，我们称之为连通图 。极大连通子图称之为该图的一个连通分量。

在有向图中，任何两个节点是可以相互到达的，我们称之为强连通图。极大连通子图称之为该图的一个强连通分量。

图的构造

邻接矩阵

邻接矩阵使用二维数组来表示图结构，是从节点的角度来表示图，有多少节点就申请多大的二维数组。

① 优点：表达方式简单，易于理解；检查任意两个节点间是否存在边，操作非常快；适合稠密图，在边数接近顶点数平方的图中，邻接矩阵是一种空间效率较高的表示方法。

② 缺点：遇到稀疏图，会导致申请过大的二维数组造成空间浪费。且遍历边的时候需要遍历整个矩阵，造成时间浪费。

邻接表

邻接表 使用 数组 + 链表的方式来表示。 邻接表是从边的数量来表示图，有多少边 才会申请对应大小的链表。

① 优点：遇到稀疏图，只需要存储边，空间利用率高；遍历节点连接情况相对容易。

② 缺点：检查任意两个节点间是否存在边，效率相对低。时间复杂度 O(V)，V 表示连接节点的数量。实现相对复杂，不易理解。

图的遍历

深度优先搜索（DFS）、广度优先搜索（BFS）。也可以在不同的数据结构上进行搜索：

二叉树的递归遍历是 DFS，二叉树的层序遍历是 BFS。而在图论章节，则是在邻接表或邻接矩阵上进行搜索。

public class Main {
    static List<Integer> path = new ArrayList<>();
    static List<List<Integer>> paths = new ArrayList<>();


    static void dfs(List<List<Integer>> graph, int i, int m){
        if(i == m){

卡码网题目链接（ACM模式）

现有一个 N × M 的矩阵，每个单元格包含一个数值，这个数值代表该位置的相对高度。 矩阵的左边界和上边界被认为是第一组边界，而矩阵的右边界和下边界被视为第二组边界。 矩阵模拟了一个地形，当雨水落在上面时，水会根据地形的倾斜向低处流动，但只能从相邻（上下左右方向）的地点流动。 我们的目标是确定那些单元格，从这些单元格出发的水既可以到达第一组边界，也可以到达第二组边界。

输入示例：

1 3 1 2 4
1 2 1 3 2
2 4 7 2 1
4 5 6 1 1
1 4 1 2 1

输出示例：

为了防止超时，我们可以反过来想：从第一组边界上的节点逆流而上，将遍历过的节点都标记上。

同样从第二组边界上的节点逆流而上，将遍历过的节点都标记上。然后两方都标记过的那些节点就是题目所需。

import java.util.Scanner;


public class Main {
    static int[][] next = {{-1, 0}, {0, -1}, {1, 0}, {0, 1}};
    static int n, m;


    static void dfs(int[][] graph, boolean[][] visited, int x, int y) {
        if (visited[x][y]) return;
        visited[x][y] = true;
        for (int i = 0; i < 4; i++) {
            int x2 = x + next[i][0];
            int y2 = y + next[i][1];
            if (x2 < 0 || x2 >= n || y2 < 0 || y2 >= m) continue;
            if (graph[x2][y2] < graph[x][y]) continue;
            dfs(graph, visited, x2, y2);
        }
    }

11.3 并查集

11.3.1 理论基础

并查集

并查集常用来解决连通性问题。大白话就是当我们需要判断两个元素是否在同一个集合里的时候，我们就要想到用并查集。

路径压缩：在实现 find 函数的过程中，通过递归的方式不断获取 father 数组下标对应的数值，最终找到这个集合的根。 搜索过程像是一个多叉树中从叶子到根节点的过程，如果高度很深的话，每次 find 函数去寻找根的过程就要递归很多次。 我们的目的只需要知道这些节点在同一个根下就可以，除了根节点其他所有节点都挂在根节点下。

模板

xxxxxxxxxx
static private int[] father;


static private void init(int n) {
    father = new int[n + 1];
    for (int i = 1; i <= n; i++) father[i] = i;
}


static private int find(int u) {
    if (u == father[u]) return u;
    else return find(father[u]);
}


static private void join(int u, int v) {
    u = find(u);
    v = find(v);
    if (u != v) father[v] = u;
}

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int m = sc.nextInt();
        List<Integer> inDegree = new ArrayList<>();
        Map<Integer, List<Integer>> map = new HashMap<>();
        List<Integer> result = new ArrayList<>();


        for (int i = 0; i < n; i++) inDegree.add(0);
        for (int i = 0; i < m; i++) {
            int s = sc.nextInt();

                map.put(s, new ArrayList<>());
            }
            map.get(s).add(t);      // map = (0,[1,2]), (1,[3]), (2,[4])
        }
        Queue<Integer> q = new LinkedList<>();
        for (int i = 0; i < n; i++) {
            if (inDegree.get(i) == 0) q.offer(i);
        }

                    inDegree.set(file, inDegree.get(file) - 1);

                }
            }
        }

                        minDist[y] = tmp[x] + price;
                        if (!visited[y]) {
                            visited[y] = true;
                            q.offer(y);
                        }
                    }
                }
            }
        }
        System.out.println(Integer.MAX_VALUE == minDist[dst] ? "unreachable" : minDist[dst]);
    }
}

import java.util.*;


public class Main {
    static int[][] moved = {{-2, -1}, {-2, 1}, {-1, -2}, {-1, 2}, {1, -2}, {1, 2}, {2, -1}, {2, 1}};
    static int endX, endY;
    static int[][] steps = new int[1001][1001];


    static class Knight {
        public int x, y;
        public int f, g, h;
    }


    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        while (n-- > 0) {
            for (int[] step : steps)
                Arrays.fill(step, 0);
            Knight start = new Knight();
            start.x = sc.nextInt();
            start.y = sc.nextInt();
            endX = sc.nextInt();
            endY = sc.nextInt();
            start.g = 0;                    // 从起点到该节点路径消耗
            start.h = distance(start);      // 该节点到终点的路径消耗
            start.f = start.g + start.h;    // 从起点到终点的路径消耗
            aStar(start);
            System.out.println(steps[endX][endY]);
        }
    }


    private static int distance(Knight knight) {
        return (knight.x - endX) * (knight.x - endX) + (knight.y - endY) * (knight.y - endY);
    }


    private static void aStar(Knight knight) {
        PriorityQueue<Knight> q = new PriorityQueue<>(Comparator.comparingInt(k -> k.f));
        q.offer(knight);
        while (!q.isEmpty()) {
            Knight node = q.poll();
            if (node.x == endX && node.y == endY) break;
            for (int i = 0; i < 8; i++) {       // 八种 "马走日"
                int x2 = node.x + moved[i][0];
                int y2 = node.y + moved[i][1];
                if (x2 < 1 || y2 < 1 || x2 > 1000 || y2 > 1000 || steps[x2][y2] != 0)
                    continue;
                steps[x2][y2] = steps[node.x][node.y] + 1;
                Knight node2 = new Knight();
                node2.x = x2;
                node2.y = y2;
                node2.g = node.g + 5;           // 每次移动 2^2 + 1^2 = 5
                node2.h = distance(node2);
                node2.f = node2.g + node2.h;
                q.offer(node2);
            }
        }
    }
}

什么时候用单调栈呢？通常是一维数组，要寻找任一个元素的右边或者左边第一个比自己大或者小的元素的位置。

单调栈的本质是空间换时间，因为需要用一个栈来记录右边第一个比当前元素高的元素，优点是整个数组只需要遍历一次。

class Solution {
    public int[] dailyTemperatures(int[] T) {
        Stack<Integer> s = new Stack<>();
        int[] result = new int[T.length];
        s.push(0);
        for (int i = 1; i < T.length; i++) {
            while (!s.isEmpty() && T[i] > T[s.peek()]) {
                // i: 1 -> 2 -> 5 -> 5 -> 6 -> 6
                int index = s.pop();        // 0 -> 1 -> 4 -> 3 -> 5 -> 2
                result[index] = i - index;  // 1 -> 1 -> 1 -> 2 -> 1 -> 4
            }

        }
        return result;
    }
}

class Solution {
    public int[] nextGreaterElement(int[] nums1, int[] nums2) {

        int[] result = new int[nums1.length];
        Arrays.fill(result, -1);
